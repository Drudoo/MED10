## Build wirelessly to Android (Mac)

- Install ``adb``:


	brew install android-platform-tools


Plug in device:

    adb tcpip 5555

Unplug device

    adb connect deviceIP

Build and Run normally from Unity.

## Get Debug.Log() from Android (Mac)

- Using ``adb``:


	adb logcat
